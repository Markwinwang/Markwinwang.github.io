01

记忆不是一个仓库，而是一组协同系统

Gateway

感觉系统

视觉、听觉、触觉与语言网络先把世界切成可处理的特征。

Control

注意系统

决定哪些信息值得进入工作台：新颖性、目标相关性、情绪强度。

Workspace

工作记忆

前额叶与顶叶把信息短暂保持，并进行比较、推理、组合。

Index

海马体

把时间、地点、人物、概念和情绪绑定成可召回的 episode。

World model

新皮层

慢速吸收、抽象、泛化，把具体经历转成长期语义知识。

Weighting

杏仁核

给恐惧、惊喜、奖赏和失败打权重，决定记忆优先级。

Policy

基底节

通过强化学习形成习惯，把高频行为变成自动化策略。

Calibration

小脑

做误差校正、节律控制和精细协调，是技能流畅化的核心。

05

把神经机制转成学习方法

Pattern separation

先切开相似概念

学习深度来自差异辨认。每学一个新概念，问：它和相似概念的边界在哪里？

  • ADC vs PDC vs RDC
  • 工作记忆 vs 短期记忆
  • 情景记忆 vs 语义记忆
Pattern completion

用线索训练召回

不要只重读。用一个关键词、一张图或一个问题，主动召回整套结构。

  • 闭卷复述
  • 画机制图
  • 用 3 个类比解释
Systems consolidation

让睡眠参与学习

白天编码，夜间离线训练。间隔重复比单次狂刷更符合大脑机制。

  • 当天理解
  • 隔天重构
  • 一周后输出成 memo 或 slide
